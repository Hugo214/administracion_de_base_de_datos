# Módulo 1: Introducción a la Administración de Bases de Datos

## Objetivos del módulo
- Entender el rol del DBA en una organización
- Conocer las responsabilidades diarias, semanales y mensuales
- Diferenciar entre tipos de bases de datos

## Teoría

### Rol del DBA
El Administrador de Base de Datos es responsable de:
- **Disponibilidad:** Que la BD esté online 24/7
- **Rendimiento:** Queries rápidas y uso eficiente de recursos
- **Seguridad:** Acceso controlado y datos protegidos
- **Recuperación:** Poder restaurar datos ante desastres

### Tareas típicas de un DBA
| Frecuencia | Tareas |
|------------|--------|
| Diarias | Monitoreo de logs, backups, espacio en disco |
| Semanales | Revisión de queries lentos, actualización de estadísticas |
| Mensuales | Pruebas de restauración, auditoría de usuarios |
| Trimestrales | Plan de capacidad, parches de seguridad |

### Tipos de Bases de Datos

#### Relacionales (SQL)
- **MySQL/MariaDB:** Web, startups, código abierto
- **PostgreSQL:** Datos complejos, compliance
- **SQL Server:** Entornos Microsoft, BI
- **Oracle:** Gran escala, misión crítica

#### NoSQL
- **MongoDB:** Documentos JSON, alta escalabilidad
- **Redis:** Caché, sesiones, tiempo real
- **Cassandra:** Escrituras masivas, distribución geográfica
